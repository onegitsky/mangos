# Keybindings
> Only basic and main keybinds. All other binds can be found in [mango](./mango/)

### Rofi
| Bind | Action |
| ---- | ------ |
| `Super` + `Z` | Open Launcher |
| `Super` + `X` | Open Powermenu |
| `Super` + `V` | Open Clipboard |
| `Super` + `H` | Open Wallpaper Selector |

### Software & Windows
| Bind | Action |
| ---- | ------ |
| `Super` + `Q` | Close Window |
| `Super` + `L` | Lock screen |
| `Super` + `Return` | Open Terminal |
| `Super` + `C` | Open Btop |
| `Super` + `Shift` + `D` | Open Nemo |
| `Ctrl` + `Alt` + `Q` | Open Browser |
| `Super` + `Shift` + `R` | Reload Mango |
| `Super` + `F` | Fullscreen Window |
| `Super` + `Shift` + `S` | Screenshot |
